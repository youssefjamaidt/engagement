# Gestionnaire d'Engagements - Natation

Site web professionnel pour gérer les engagements des nageurs lors des compétitions.

## Fonctionnalités

- ✅ Gestion des nageurs et de leurs engagements
- ✅ Tableau interactif avec cases à cocher
- ✅ Statistiques en temps réel
- ✅ Exportation des données en CSV
- ✅ Sauvegarde automatique dans le navigateur
- ✅ Design responsive et moderne

## Utilisation

1. Ouvrir `index.html` dans un navigateur web
2. Ajouter/modifier les nageurs si nécessaire
3. Cochez les cases pour les épreuves de chaque nageur
4. Exportez les données avec le bouton "Exporter"

## Technologies

- HTML5, CSS3, JavaScript vanilla
- Local Storage pour la sauvegarde
- Design responsive
- Pas de dépendances externes
